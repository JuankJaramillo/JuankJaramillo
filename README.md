- Creación de un servidor local por medio de Ubuntu y en la nube con ayuda de AWS
- Montaje funcional de un ciclo Rankine experimental
- Control de múltiples brazo robotics por medio de Matlab
- Calculo, simulación,  diseño e Implementación de un biodigestor con separación de biol y compost,  con su posterior empaquetado automatizado
